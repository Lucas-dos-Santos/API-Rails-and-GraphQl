## Mutations
  * createUser(input: CreateUserInput)
  * deleteUser(input: DeleteUserInput)
  * updateUser(input: UpdateUserInput)

### A simple query
```
  query {
  users{
    id
    name
    email
    booksCount
    books {
      title
    }
  }
}
```
